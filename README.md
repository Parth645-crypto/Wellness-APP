# 🌿 Rooted – Your Personal Growth Ecosystem

Rooted is a personalized self-growth app that transforms your daily habits and emotions into a living digital ecosystem.

Instead of just tracking habits, Rooted creates a feedback loop where:
- Your daily rituals earn XP
- Your level increases
- Your ecosystem evolves visually
- Your mood influences future growth

It combines habit tracking, gamification, and emotional awareness into one adaptive system.

## ✨ Features

- 🌱 Personalized onboarding based on user lifestyle
- 📋 Daily ritual system (auto-generated tasks)
- 🎯 XP & Level progression system
- 🌳 Visual ecosystem growth (tree evolution)
- 😊 Mood tracking with environmental feedback
- 🔁 Daily reset & adaptive ritual generation
- 📊 Ecosystem health breakdown (physical, mental, sleep)
- 👤 Profile & baseline tracking

## 🧩 How It Works

1. User completes onboarding → sets lifestyle baseline  
2. App generates personalized daily rituals  
3. Completing rituals → earns XP  
4. XP → increases level → evolves ecosystem  
5. Mood selection influences future environment  

Rooted creates a continuous self-improvement loop.

## 🛠️ Tech Stack

- SwiftUI
- MVVM Architecture
- Combine (for state management)
- UserDefaults / AppStorage (for persistence)

## 🎯 What Makes It Unique

Unlike traditional habit trackers, Rooted:
- Adapts to user lifestyle instead of using fixed tasks
- Connects emotions with system behavior
- Uses visual ecosystem growth as feedback
- Focuses on long-term behavioral change rather than daily checklists

- Cloud sync (Firebase / iCloud)
